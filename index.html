get to know the person behind the code

1998

Halo! Saya Gark2091, lahir di era transisi teknologi dari dial-up ke fiber optic, dari BIOS ke UEFI, dari single-core ke 16-core. Saya tumbuh bersama evolusi digital dan jatuh cinta pada dunia teknologi sejak pertama kali membongkar komputer Pentium 2 milik ayah saya.

2015

Memulai perjalanan coding dengan belajar HTML dan CSS di warnet sambil main game. Kemudian berkembang ke JavaScript, PHP, dan akhirnya menemukan passion di Python. Membuat project pertama : website sederhana untuk komunitas game lokal.

2018

Masuk ke dunia cybersecurity setelah website pertama saya di-hack. Dari situ belajar tentang ethical hacking, penetration testing, dan keamanan web. Bergabung dengan komunitas CTF dan mulai mengikuti kompetisi lokal maupun internasional.

2020

Pandemi jadi momentum untuk deepening skills. Fokus ke reverse engineering dan binary exploitation. Membangun homelab sendiri untuk praktek, lengkap dengan server, firewall, dan berbagai perangkat jaringan untuk simulasi.

2022

Memulai perjalanan di bug bounty, menemukan beberapa vulnerability di aplikasi terkenal. Juga aktif sebagai mentor di komunitas cybersecurity lokal, membantu pemula belajar ethical hacking dengan cara yang bertanggung jawab.

2024

Sekarang fokus ke blockchain security dan smart contract auditing. Percaya bahwa web3 akan jadi masa depan, tapi butuh foundation security yang kuat. Juga mulai menulis blog dan membuat konten edukasi tentang cybersecurity.
